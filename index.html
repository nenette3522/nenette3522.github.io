Portfolio d'une Nénette

Ce site a pour but de pouvoir présenter les différents projets réalisés pour les cours que je suis sur Codecademy.

  • Guide de styles : un guide de styles à appliquer en CSS.
  • La Cafeterie : page d'accueil d'un café immaginaire, responsive.
  • The Silly Game Sentence : À la base ça devait être un projet qui devait se passer dans la console, sauf que chez CodeCademy ils ont appelé ce projet "Projet Portfolio". Sauf que trois bout de code que j'arrive à faire tourner dans la console, je trouve pas ça ouf de ouf, je peux pas vraiment le partager, c'est pas fou pour quelque chose qui est censé être un projet portfolio. Alors je me suis mis un challenge, faire l'exercice demandé avec une interface graphique (vous aurez deviné qu'il s'agit donc du navigateur), et puis tant qu'à faire que le joueur puisse composer lui-même sa Silly Sentence. Le programme pioche aléatoirement un mot de chaque type dans une liste. Ce projet est le premier où je réalise moi-même de A à Z une page web interactive avec JavaScript. Et c'est pas pour me lancer des fleurs (si), mais cette page est également responsive.

C'est vrai que cette page-ci n'est pas ouf. Je le reconnais. Je le sais. C'est un work in progress. J'avoue que je passe beaucoup plus de temps sur les exercices et les projets pratiques que sur cette page. J'ajoute les projets dans l'ordre au fur et à mesure que je les fais. Ça va me permettre de voir les progrès que j'ai fait. Cela va faire 4 mois que j'ai commencé, et j'ai fait 35% du carreer path en Data Science (j'aime beaucoup Python), et j'en suis à peu près au même niveau de complétion pour le carreer path ingénieur fullstack. J'ai aussi obtenu 7 certificats de complétion de cours de CodeCademy.
